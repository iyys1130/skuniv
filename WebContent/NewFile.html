트렐로란

  • 나를 계획적인 사람으로 만들어주는 협업 보드. 같은 업무에 참여하는 동료들이 업무 진행상황을 확인하고 의견을 나눌 수 있도록 도와주는 응용 프로그램, 구글 계정으로 무료 이용 가능
  • URL : http://trello.com

가입 하기

가입 프로세스 개요

  1. 크롬 브라우저에서 본인의 아이디(ID@skuniv.ac.kr)로 로그인

  2. 크롬 브라우저에서 새 탭을 열고 트렐로 웹페이지(http://trello.com)에 접속

    1. 오른쪽 상단 Sign Up 버튼 클릭

    2. Sign up with Google 버튼 클릭

    3. Login with Google 버튼 클릭

    4. Allow 버튼 클릭

    5. Create a new Trello Account 클릭

로그인 프로세스 개요

  1. 크롬 브라우저에서 본인의 아이디(ID@skuniv.ac.kr)로 로그인

계정 이름 바꾸기



계정 이름 바꾸기 프로세스 개요

  1. 로그인 후 오른쪽 상단 내 이름 클릭

  2. Profile 클릭 → Edit Profile 클릭

  3. Full name에 본인 이름 (Ex 백빛나) User name에 본인 아이디 (Ex bitnabaek) 입력

    1. 본인 아이디는 @skuniv.ac.kr 앞에 있는 아이디명을 의미함

    2. 아이디를 바꾼 뒤 본인 아이디명을 캐스터즈 담당자에게 전달, 트렐로 보드에 초대해달라고 요청

보드 사용하기

보드 사용법 개요

  • SKU CAST 2nd Board에서 서포터즈 2기 프로젝트 활동을 진행

  • 트렐로 소개 관련 뉴스기사

    • 트렐로 : 체계적인 사람으로 거듭나는 방법 [뉴스기사 바로가기]

    • [직장인앱-'트렐로'] 업무별 세부 작업 메모지 붙이듯 표시… 동료들간 의견도 공유 [뉴스기사 바로가기]

    • 본인 팀의 리스트안에 프로젝트별로 카드를 생성해 업무 진행상황 공유 [보드 > 리스트 > 카드] 순 계층화



보드에 리스트 생성하기





보드에 카드 생성하기

  1. 카드를 생성해야할 리스트 하단 Add a card 클릭

  2. 카드 제목 입력

    1. 카드 제목은 프로젝트별로 수행할 수 있도록 함

    2. Ex) 서경대학교 호기심천국 시리즈, 캠퍼스 패션 IN SKU 시리즈 등

    3. 카드가 생성되면 해당 카드의 용도를 Edit the description을 눌러 내용 추가 또는 수정







보드에 멤버 추가하기

  1. 메뉴 > Add Members 클릭
  2. 사용자 이름 한글로 검색 후 아이디 확인 후 추가하기 (추가하기 전 반드시 확인해야 할 사항 : 추가할 대상이 트렐로에 가입되어있는지? Full name, User Name을 제대로 변경했는지?)

마크다운 문법

  • 마크다운이란? [자세히 알아보기]
    • 마크다운(markdown)은 일반 텍스트 문서의 양식을 편집하는 문법이다. README 파일이나 온라인 문서, 혹은 일반 텍스트 편집기로 문서 양식을 편집할 때 쓰인다. 마크다운을 이용해 작성된 문서는 쉽게 HTML 등 다른 문서형태로 변환이 가능하다
    • . - 출처 위키백과 -
    • 트렐로에서 자주쓰는 마크다운 문법
    • 제목
      • #제목 수준1
      • ##제목 수준2
      • ###제목 수준3
      • ####제목 수준4
    • 굵게, 기울이기 :
      • **굵게**
      • _기울이기_ 또는 *기울이기*
    • 인용문
      • >인용할 문구
    • 링크
      • [링크제목](URL주소)
    • 가로선
      • ---
    • 취소선
      • ~~취소선~~

      • - ● 목록
      • 1. 숫자목록
